# User Story: Inschrijfdatum als eerste kolom

Als beheerder wil ik in de aanmeldingenlijst bij een activiteit de inschrijfdatum als eerste kolom zien in het formaat `dd/mm/jjjj`, zodat ik snel kan zien wanneer iemand zich heeft aangemeld.

## Acceptatiecriteria
- De aanmeldingenlijst toont als eerste kolom de inschrijfdatum.
- De datum wordt weergegeven in het formaat `dd/mm/jjjj`.
- De bestaande kolommen Naam, Telefoon en Mail blijven zichtbaar.
- De wijziging geldt voor de activity detail weergave en de lijst die in de mail wordt gebruikt, indien van toepassing.

## Opmerking
- De datum mag gebaseerd worden op `created_at` van de aanvraag.
